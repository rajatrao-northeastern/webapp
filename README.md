## Installation

1. Clone the repository.
```bash
git clone <repo SSH protocol>
```

2. Navigate to project directory.
```bash
cd <path to cloned directory>
```

3. Run 
```bash
npm install
```

4. Install Browser/Postman to view running application.
```bash
npm start
```

5. Verify API Endpoints

1. /api
2. /api/user
3. /api/user/kjkj
